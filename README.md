# Smart Currency Denomination Detection System

An AI-powered, cross-platform assistive application designed to help users, particularly individuals with visual impairments, identify Ghanaian currency denominations through real-time camera scanning or captured images.

## Project Structure

- `mobile/`: Flutter application for mobile (Android/iOS) and cross-platform shared code.
- `web/`: Web-specific deployment or alternative flutter web entry point.
- `ai/`: Python-based model training environment and notebooks.
- `dataset/`: Training images for the AI model.
- `docs/`: Project documentation.
- `backend/`: (Future) Backend API.
